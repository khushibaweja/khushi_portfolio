## Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Git installed on your system
- GitHub account

### Installation

1. Clone the repository:
```bash
git clone https://github.com/khushibaweja/portfolio.git
```

2. Navigate to the project directory:
```bash
cd portfolio
```

3. Add your profile image named `khushi.jpg` in the root folder

4. Open `index.html` in your browser or use a local server:
```bash
python -m http.server 8000
```

## Deploying to GitHub Pages

### Step 1: Create GitHub Repository

1. Login to GitHub
2. Create a new repository named `portfolio` or `yourusername.github.io`
3. Set visibility to public
4. Do not initialize with README

### Step 2: Push Your Code

```bash
git init
git add .
git commit -m "Initial portfolio commit"
git remote add origin https://github.com/yourusername/portfolio.git
git branch -M main
git push -u origin main
```

### Step 3: Enable GitHub Pages

1. Go to repository Settings
2. Navigate to Pages section in sidebar
3. Under Source, select "main" branch
4. Click Save
5. Wait 1-2 minutes for deployment
6. Access your site at: `https://yourusername.github.io/portfolio/`

## Customization

### Changing Colors

Edit CSS variables in the style section:

```css
:root {
    --primary: #a855f7;
    --secondary: #ec4899;
    --accent: #3b82f6;
    --dark-bg: #0a0a0f;
}
```

### Updating Content

Modify the HTML sections directly in `index.html` to update personal information, projects, skills, and contact details.

### Adjusting Animations

Modify JavaScript variables:
- `typingDelay` and `erasingDelay` for typing speed
- `duration` in smooth scroll function
- `particlesJS` configuration for particle effects
